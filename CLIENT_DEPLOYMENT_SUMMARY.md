## 🌐 Live Features

### Search Functionality
- Type any disease name, symptom, or keyword
- Results update instantly as you type
- Search across all 5 information fields
- Sorted by relevance (name matches first)

### Example Searches
- "autism" → Autism Spectrum Disorder
- "anxiety" → Anxiety Disorder, Generalized Anxiety
- "diabetes" → Diabetes Type 1, Type 2, Gestational
- "depression" → Depression, Postpartum Depression
- "sleep" → Sleep Disorders, Insomnia

### Design Elements
- Dark brown background (#171515) for easy reading
- Gold accents (#d4af37) matching Berashith branding
- Professional footer with full attribution
- Header image with full-width banner
