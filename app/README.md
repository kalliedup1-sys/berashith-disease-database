# Disease & Sickness Database Search Engine

A comprehensive disease and sickness database search application built for Berashith Ministry.

## Features

- **248 Diseases** - Complete database of diseases and sicknesses
- **Real-time Search** - Search as you type across all fields
- **5-Field Information** - Each disease includes:
  - Name
  - Description
  - General Information
  - Roots/Symptoms
  - Recommendations

## Design

- Dark theme (#171515) with gold accents (#d4af37)
- Professional, clean interface matching Berashith Ministry branding
- Full-width responsive design
- Mobile-friendly

## Technology

- Pure HTML5, CSS3, and Vanilla JavaScript
- No external dependencies
- Fast, lightweight, and reliable

## Installation

1. Clone this repository
2. Open `app.html` in a web browser
3. Start searching for diseases

## Deployment

Hosted at: https://berashithministry.co.za/app
